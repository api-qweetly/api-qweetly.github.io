How to setup QweetlyAPI

First make a issue on https://github.com/api-qweetly/api-qweetly.github.io/issues/ after that make a issue named "API intergration with my project" DO NOT add the project name and thesite in the title, only in the description in the description you say what of our api services you want to use, but make sure to include the site name in the description

After that now wait until we respond to your request and we will allow our API to be used on your site.
